PROBLEM: demo.kisag.co was redirecting to demo.kisag.co/dashboard (ugly URL)

SOLUTION: Changed NextResponse.redirect() to NextResponse.rewrite()

RESULT:
✅ demo.kisag.co → Shows dashboard at root (clean URL)
✅ kisag.co → Shows main marketing site
✅ URL stays clean (no /dashboard visible)

FILES CHANGED:
- /middleware.ts → Uses rewrite() instead of redirect()
- /vercel.json → Removed conflicting rewrites

KEY CODE:
  return NextResponse.rewrite(url);  // ✅ Keeps URL same
  // NOT
  return NextResponse.redirect(url); // ❌ Changes URL

TESTING:
  Local:  http://demo.kisag.localhost:3000
  Prod:   https://demo.kisag.co

DEPLOYMENT:
  1. git add . && git commit -m "Fix multi-domain routing" && git push
  2. Add demo.kisag.co in Vercel dashboard
  3. Configure DNS: CNAME demo → cname.vercel-dns.com
  4. Wait 5-15 minutes
  5. Test: curl https://demo.kisag.co
